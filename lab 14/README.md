Цель работы

Освоить методы тестирования и отладки программ на языке Python. Научиться использовать модуль unittest для автоматического тестирования и модуль logging для логирования ошибок.

⸻

Задание

Разработать программу для расчёта итоговой оценки студента.
Программа должна:
	•	принимать список оценок,
	•	вычислять средний балл,
	•	определять буквенную оценку (A, B, C, D, F),
	•	логировать ошибки,
	•	содержать unit-тесты,
	•	иметь файл README.

⸻

В ходе выполнения лабораторной работы были использованы:
	•	Python — основной язык программирования

Ход работы

1. Реализация основной программы

Была создана программа main.py, содержащая функции:
	•	calculate_average() — вычисляет средний балл,
	•	determine_grade_letter() — определяет буквенную оценку,
	•	student_report() — формирует отчёт.

Также настроено логирование в файл app.log.

⸻

2. Логирование

Для записи ошибок и событий был использован модуль logging.

Логируются:
	•	пустой список оценок,
	•	неверный тип данных,
	•	успешные операции,
	•	исключения.

Пример сообщений в log-файле:
	•	Ошибки ввода данных
	•	Успешный расчёт среднего балла
	•	Создание отчёта студента

⸻

3. Модульное тестирование

Создан файл test_main.py, в котором реализованы unit-тесты:

Тест	Назначение
test_average_calculation	Проверка расчёта среднего
test_average_empty	Проверка пустого списка
test_average_type_error	Проверка неверных типов
test_letter_grade	Проверка буквенных оценок
test_student_report	Проверка формирования отчёта

Тесты успешно выполняются командой:

python test_main.py
